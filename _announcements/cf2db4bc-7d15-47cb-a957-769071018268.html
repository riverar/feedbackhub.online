Updated: 12/13 Resolved Issue – Windows 10 Mobile Release Preview Build 14393.576

Wallet tap to pay capabilities have been restored for Release Preview users on OS build 14393.576.  If you are on the Release Preview ring confirm that the Payment feature is now available again.  You may need to re-add your credit cards to use tap to pay.  Please send user feedback if you continue to experience any issues using tap to pay.

Known Issue

Wallet tap to pay capabilities have been disabled due to a bug in OS build 14393.479. 

Tap to pay will be fixed in the next Release Preview OS flight. You will need to re-add any credit cards that have been deleted.

We are sorry for any inconvenience this has caused and hope that you will continue to use Wallet for tap to pay.
