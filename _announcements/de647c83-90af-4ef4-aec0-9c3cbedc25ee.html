Windows Insiders with the latest Windows 10 Insider Preview build running on their phones will be able to try out Skype Preview app which is currently rolling out. This release brings one of the most requested Skype features to your Windows 10 phone: the ability to make group video calls.

Later this summer we will bring new features to Skype on Windows 10 Mobile, including SMS relay. With this feature, you’ll be able to send SMS messages from a Windows 10 PC or Windows 10 phone, giving you the ability to send and receive texts, group messages, and photos all in one app, as well as a single view of your SMS and Skype conversations.

The Skype Preview app is getting updated for PC as well to 11.6.187.0 which brings some new features including the ability to call mobiles and landlines at low rates, leave and receive voicemails, added additional collaboration tools such as file sharing, screen sharing and translation for audio and video calls and a preview of Skype Bots. This update also brings many improvements overall app performance and experience.

For more information – please read Skype’s blog post.
