For Windows Insiders in the Fast ring, we are rolling out an update to the Windows Store that brings some improvements designed to make the install experience smoother - especially for large PC games.

New pick a drive experience for large PC games

For large games the store now allows the customer to pick their desired drive for download and install putting the customer in the driverâ€™s seat.

If a certain app/ game does not fit on the default drive, the store pre-emptively asks the customer to pick a different drive. This prevents users from getting in an error state.

New Install control

The install experience has been improved. Customers now see the exact progress of their install and the install control button changes shade.
