Today we have released an Cumulative Update (KB3154879) for Windows 10 Insider Preview Build 14295. This build is currently in the Slow ring.

This update contains select fixes based upon user feedback, including:

  • Battery life improvements for devices with TPM hardware.
  • Reduced power usage and enhanced battery life for Surface Book devices.
  • Edge browser stability improvements for websites running JavaScript.
  • Reliability improvements to remedy install failures of Visual Studio 2015.
