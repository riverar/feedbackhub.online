--- layout: page title: 'PIN and Windows Hello might stop working after taking a new flight' contentId: 7516b1ba-8e0c-450e-8858-f31a94531b56 date: 2015-07-31T00:00:00-07:00 ---

Build 10149 was the first build we released externally to Insiders after we made some changes to our Passport infrastructure. Some users may find that their PIN and Windows Hello has stopped working once they receive an automated build upgrade to Build 10149 (or higher).

If this happens to you, reset your PIN by going to Settings > Accounts > Sign-in options and clicking on “I forgot my PIN”. Note that the PINs for all users on the device must be reset this way for any PIN or Windows Hello to start working again.
